Hugo, 개발 절차

  1. Hugo 설치
apt install hugo
  1. 프로젝트 생성
hugo new site <site-name>
  1. 테마 사용
git clone <theme-url>
  1. 파일 만들기
hugo new <page-name>
  1. 테스트
hugo server --bind=0.0.0.0
  1. 빌드
hugo -t <테마이름>:hugo-developer-portfolio
  1. 퍼블리싱
cd <public 경로>
git add .
git commit -m "<commit-message>"
git push
